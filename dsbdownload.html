F.A.Q

Frequently Asked Questions

  • What is Dual Status Bar aka DSB ?

    Dual Status Bar, or DSB, is a Magisk module which can add a second row to both sides of your statusbar.

  • Which rom and devices support DSB ?

    Here you have a list of ROMs which we support right now :

    • AOSP: Android 10,11,12
    • MIUI: Android 10 (use our substratum app) & A11
    • OOS: Android 10
    • Other stock based roms also supported, ask in group with screenshot

  • Is DSB available for my device? How to find my DSB file ?

    • To download a compatible DSB file go to our Telegram group and the use note #download (or click here) to see the list of general DSB and device specific DSB, you have to check both lists!
    • Once you've found a file for your device and rom check the pling link to download the module If you get difficulty finding on pling use the note #how-to-download (or click here).

  • How to flash and enable my dsb file ?

    Flash the zip directly on Magisk, then reboot.
    Read the note #enable-dsb in support group for detailed steps about how to enable it after rebooting (or click here)

  • What do to if my dsb is not present under download message ?

    Here You have two options :

    • Use #request-format (or click here) to request for a new dsb on our support group, any admin will take your request when he's available.
    • Look for a different device on your same rom version, and give a try for that other file. As long as your rom version or build date is the same you have good chances that It'll work too.

    If you try this second method check #bootloopfix (or click here) to be able to delete the module in case you hace any issues. Also please report us if you find a module from another device working on yours!

  • Is a recovery version of DSB available ?

    No, there are no zip files to flash in recoveries like TWRP or OrangeFox.
    We only build Magisk modules since they’re easier to remove in case if any issue happens.

  • My DSB module is not working !

    Check this Telegraph Post that we’ve made collecting every issue and its fix.

  • Why should I Donate ?
